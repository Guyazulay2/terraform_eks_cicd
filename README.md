### install tf first :

```
sudo apt-get update && sudo apt-get install -y unzip
wget https://releases.hashicorp.com/terraform/1.9.4/terraform_1.9.4_linux_amd64.zip
unzip terraform_1.9.4_linux_amd64.zip
sudo mv terraform /usr/local/bin/
terraform -v
```
